# [EcoTrail](https://eco-trail-5db29.web.app/index.html)
EcoTrail is a mobile-first web application designed to promote eco-friendly habits by helping users track, understand, and reduce their carbon footprint by opting for greener transport options.

- Tracks daily commute and the carbon footprint generated based on the type of transport used (public/private)
- Encourages sustainable choices through incentives, like vouchers, and community challenges
- Promotes environmental awareness using data-driven feedback
